Application Context vs Bean Factory:

Application context thì default là EAGER, gọi tất cả các bean
Bean Factory thì default là LAZY mỗi lan muốn gọi bean thì phải dùng phương thức getBean
Bean Factory chỉ dùng xml
Apllication thì dùng 3 cái
